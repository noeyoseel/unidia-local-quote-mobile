# [유니디아] 로컬 PC 기반 자동차 반자동 견적 자동화 시스템 통합 기획 사양서

## 1. 프로젝트 개요
* **시스템 명칭**: 유니디아 로컬 반자동 견적 비교 시스템 (Unidia Local Semi-Automatic Quote System)
* **기획 배경**: 
  * 금융 수식 및 잔가율 정책이 복잡하게 얽힌 캐피탈사별 엑셀 계산기를 외부 웹 서버나 구글 시트로 강제 변환할 경우, 금융 함수(IRR, PMT, RATE)가 깨지거나 연산 오차가 발생하는 치명적인 리스크가 존재함.
  * 매달 캐피탈사별로 금리가 변동되어 배포되는 최신 엑셀 템플릿을 현업 담당자가 직접 직관적으로 교체하고 관리할 수 있는 유연한 체계가 필요함.
  * 완전 자동화 시 발생할 수 있는 허위 견적 발행 및 금융소비자보호법 배포 위반 리스크를 방지하기 위해, 상담사의 영업적 판단(할인율, 인센티브, 마진 조정)을 결합한 '반자동(Human-in-the-loop)' 시스템을 구축하고자 함.
* **핵심 가치**: "기계는 반복적인 조건 입력, 파일 매칭, 초안 산출을 담당하여 속도를 극대화하고, 사람은 최종 영업적 판단과 오차 검수를 담당하여 무결성을 확보한다."

---

## 2. 시스템 아키텍처 및 기술 스택
외부 클라우드 서버나 외부 자동화 툴(n8n 등)을 사용하지 않고, 보안과 금융 규제 준수를 위해 오직 담당자의 윈도우 PC 내부 환경에서 독립적으로 구동되는 로컬 소프트웨어 프레임워크를 채택합니다.

* **사용자 인터페이스 (UI)**: `Streamlit` (파이썬 기반의 직관적이고 가벼운 로컬 웹 애플리케이션 프레임워크, 모바일 뷰포트 대응)
* **연산 백엔드 (Backend)**: `Python` (이미지 텍스트 파싱, 메타데이터 대조, 흐름 제어 전체 담당)
* **엑셀 연동 엔진 (Excel Engine)**: `xlwings`
  * 윈도우 PC에 설치된 '진짜 Microsoft Excel'을 백그라운드에서 직접 실행(COM 인터페이스)하여 데이터를 쓰고 읽음으로써, 캐피탈사 고유의 금융 수식을 원본 그대로 100% 보존.
* **로컬 데이터베이스 (DB)**: `SQLite` (파일 형태로 저장되는 경량 DB, 상담 이력 및 캐피탈사 파일 메타데이터 관리용)
* **로컬 디렉토리 구조**:
  ```text
  📁 Unidia_Quote_App/
  ├── 📄 app.py                  (웹 응용프로그램 구동 및 UI 제어 메인 소스)
  ├── 📁 templates/              (매달 업데이트되는 캐피탈사별 원본 엑셀 보관 폴더)
  │   ├── 📁 2026_07/
  │   └── 📁 2026_06/
  └── 📄 unidia_history.db       (로컬 상담 이력 및 데이터베이스 파일)
